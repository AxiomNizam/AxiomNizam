### 📝 Configuration Files Needed

Create after API server implementation:

**`config/api-server.yaml`**
```yaml
server:
  port: 8080
  tlsPort: 8443
database:
  type: postgres
  host: localhost
  port: 5432
  name: axiom_nizam
auth:
  type: jwt
  secretKey: ${AUTH_SECRET}
cache:
  type: redis
  host: localhost
  port: 6379
```

**`config/policy-engine.yaml`**
```yaml
engines:
  - type: cel
    enabled: true
  - type: rego
    enabled: true
  - type: dsl
    enabled: true
defaultEngine: cel
```

**`config/storage.yaml`**
```yaml
primary:
  type: postgres
  connection: postgresql://user:pass@localhost:5432/axiom_nizam
backup:
  type: s3
  bucket: axiom-nizam-backups
  region: us-west-2
```
