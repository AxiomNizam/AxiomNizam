```bash

./axiomnizamctl metrics collect
```

```bash
# All tests
go test ./internal/integration/... -v

# Benchmarks
go test ./internal/integration/... -bench=. -benchmem

go test ./internal/integration/... -v -run TestFullIntegration
```

### 🔗 Integration Points

The integration layer connects:

1. **Data Mesh** ↔ **API Banks**
   - Unified search interface
   - Cross-platform discovery
   - Consistent catalog

2. **Compliance** ↔ **All Operations**
   - Every operation audited
   - Policy evaluated
   - Results tracked

3. **Monitoring** ↔ **All Systems**
   - Health tracked
   - Metrics collected
   - Alerts generated

4. **Events** ↔ **All Components**
   - Changes recorded
   - Notifications sent
   - Audit trail maintained
