AxiomNizam

Enterprise Data Control Plane — Helm Repository

CHART v0.1.0

Quick Start

1

Add the Helm repository

helm repo add axiomnizam https://shafiunmiraz0.github.io/AxiomNizam/charts Copy
2

Update your local chart cache

helm repo update Copy
3

Install the chart

helm install axiomnizam axiomnizam/axiomnizam \ -n axiomnizam --create-namespace \ --set postgresql.password=YOUR_PASSWORD \ --set iam.sysadminPassword=YOUR_PASSWORD Copy

Zero Trust Security

TLS everywhere, risk-based auth, MFA enforcement, RBAC, field-level encryption, and continuous verification.

Bundled PostgreSQL

Self-contained StatefulSet with init scripts. Disable for external databases with one flag.

Persistent Storage

4 PVCs for Raft consensus, object storage, TLS certs, and query logs. All configurable.

SafeGate Scanner

Built-in 6-stage file scanning pipeline: metadata, MIME, SVG XSS, macros, archive bombs, and native AV.

Gatekeeper 2FA

TOTP-based MFA with risk engine, trusted devices, backup codes, and step-up challenges.

Micro-Segmentation

Optional K8s NetworkPolicies and cert-manager integration for production-grade network isolation.
